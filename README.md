# 🛍️ Customer Shopping Behavior Analysis

## 📌 Project Overview

Customer Shopping Behavior Analysis is a Data Analytics project focused on understanding customer purchasing patterns, spending habits, product preferences, and subscription behavior using transactional retail data.

The project combines **Python, MYSQL, and Power BI** to perform data cleaning, exploratory data analysis, business intelligence reporting, and interactive dashboard creation.

The objective is to generate actionable business insights that can help organizations improve customer retention, optimize marketing strategies, and increase revenue.

---

## 🎯 Business Problem

Retail businesses generate large amounts of customer transaction data. Without proper analysis, valuable insights remain hidden.

This project aims to answer key business questions such as:

* Which customer groups generate the most revenue?
* Which products receive the highest ratings?
* How effective are discounts in driving sales?
* Are subscribers more valuable than non-subscribers?
* Which age groups contribute the most revenue?
* What products perform best within each category?

## 🔍 Data Cleaning & Preprocessing

### Steps Performed

### 1. Missing Value Treatment

* Identified null values in the Review Rating column.
* Imputed missing values using the median rating of each product category.

### 2. Column Standardization

* Converted column names into snake_case format.

### 3. Feature Engineering

Created new features:

* `age_group`
* `purchase_frequency_days`

### 4. Data Consistency Validation

* Checked redundancy between discount and promo code usage.
* Removed unnecessary columns.

### 5. Database Integration

* Loaded cleaned data into PostgreSQL for SQL-based business analysis.

## 💡 Key Insights

### Customer Segmentation

* Loyal customers represent the largest customer segment.

### Product Performance

* Gloves, Sandals, Boots, Hat, and Skirt received the highest average ratings.

### Shipping Preferences

* Express shipping users showed slightly higher average spending.

### Age-Based Revenue

* Young Adults generated the highest revenue contribution.

### Subscription Behavior

* Most revenue comes from non-subscribers, indicating opportunities to improve subscription adoption.
